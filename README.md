# web-app-serial-reader
reading data from arduino/esp controllers <br />
AKA digital shadow

## how to use
copy the repo <br />
change your path to ports (in my case 'COM10') <br />
`npm i` to install dependencies <br />
`npm start` to run the application http://localhost:3000
